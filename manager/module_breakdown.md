# 系统模块拆解（R2）

## M1 文档摄取模块
- 职责：读取 3 个 PDF，抽取文本并分块，建立索引
- 输入：项目目录 PDF 文件
- 输出：索引数据（chunks + embedding + metadata）
- 分类：二次开发模块

## M2 检索模块
- 职责：向量召回 + 关键词兜底 + 多文档覆盖统计
- 输入：用户问题、索引
- 输出：证据列表（含文档名、页码、片段）
- 分类：二次开发模块

## M3 回答生成模块
- 职责：基于证据生成结构化回答，严格拒答无依据问题
- 输入：问题 + 检索证据
- 输出：结论/依据/来源结构
- 分类：二次开发模块

## M4 用户视角适配模块
- 职责：医生版（专业）与患者版（通俗）回答模板
- 输入：同一证据
- 输出：不同用户视角答案
- 分类：未来可替换模块

## M5 自动出题与评测模块（Codex CLI）
- 职责：读取 3 PDF 出题、提交问答、对照评估
- 输入：PDF内容、系统回答
- 输出：eval/eval_report.md 与 eval/improvement_todo.md
- 分类：直接复用 + 脚本化模块

## M6 协作调度模块（watcher）
- 职责：识别状态并触发 manager/reviewer
- 输入：handoff/latest_status.json
- 输出：调度日志与触发动作
- 分类：二次开发模块
