# 验收标准模板

## 验收对象
- 对应任务：TASK-YYYYMMDD-001
- 审查轮次：R1

## 验收项

### 1. 功能是否完成
- [ ] 目标功能按任务单完成
- [ ] 核心流程可执行

### 2. 文件是否落地
- [ ] 任务要求创建/修改的文件都存在
- [ ] 内容完整且结构合理

### 3. 状态是否更新
- [ ] `handoff/latest_status.json` 已更新
- [ ] `stage` 与 `needs_review/needs_fix` 状态一致

### 4. 是否可进入下一阶段
- [ ] 可进入下一阶段（通过）
- [ ] 或明确进入修复阶段（needs_fix）
